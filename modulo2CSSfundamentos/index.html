  2. Displays:

    Respuesta

    display: block, El elemento con esta propiedad es tratado como un bloque. Si se adiciona otro elemento con esta misma propiedad, ese elemento "bloque" será posicionada en otra línea.

    display: inline El elemento se muestra en línea. Sin importar el tamaño que yo le dé al elemento (i.e, width:100px;height: 100px) este se mostrará en línea, ignorando el tamaño que se le asigne.

    display: inline-block Elementos con esta propiedad serán bloques mostrados en la misma línea.

    display: none, NO muestra el bloque div

    Ejemplos:

    display: none

    display: block

    display: block

    display: inline

    display: inline

    display: inline-block
    display: inline-block
