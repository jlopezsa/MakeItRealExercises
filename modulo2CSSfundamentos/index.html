  2. Displays:

    Respuesta

    display: block, El elemento con esta propiedad es tratado como un bloque. Si se adiciona otro elemento con esta misma propiedad, ese elemento "bloque" será posicionada en otra línea.

    display: inline El elemento se muestra en línea. Sin importar el tamaño que yo le dé al elemento (i.e, width:100px;height: 100px) este se mostrará en línea, ignorando el tamaño que se le asigne.

    display: inline-block Elementos con esta propiedad serán bloques mostrados en la misma línea.

    display: none, NO muestra el bloque div

    Ejemplos:

    display: none

    display: block

    display: block

    display: inline

    display: inline

    display: inline-block
    display: inline-block

  5. Ajustando imágenes:

    Ejemplos y respuestas

    Solamente con estilos width: 100px; height: 100px

    Respuesta

    Para este ejercício la figura 100x100px se incluye dentro de la etiqueta div. Este div también tiene un tamaño de 100x100px. La figura se ubica dentro de la etiqueta y mantiene su tamaño de 100x100px

    Cuando adicionamos el padding de 10px se crea un espacio entre la margen y el contenido, en este caso la figura.

    a) Adicionando el estilo padding: 10px

    Respuesta

    Se adiciona un padding de 10px al modelo de caja. La figura continúa manteniendo un tamaño de 100x100px.

    b) Adicionando el estilo padding: 10px y border: 5px

    Respuesta

    Al ejemplo anterior se adiciona un borde de 5px al modelo de caja. Así, ya tenemos el border + padding + content (figura)

    c) Cambiar el modelo de caja a border-box

    Respuesta

    Al ejemplo anterior se adiciona el estilo box-sizing: border-box. En este caso el "content", del modelo de caja, se reduce a 70x70px y la figura sale del modelo de caja manteniendo su tamaño de 100x100px. Se puede concluir que la figura sale del "content" ya que no cabe en este, es como si experimentara un desbordamiento

    En este caso, considerando un eje, el content(70px) + padding(10px + 10px) + border(5px + 5px) = 100 px

    d) Centrando la caja utilizando el valor auto en los márgenes horizontales

    Respuesta

    En este ejemplo, el modelo de caja continua teniendo un tamaño total de 100x100px. La margen, izquierda y derecha, aumenta considerablemente para centrarse a la página.
